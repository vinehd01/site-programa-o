TAGS HTML

tag html: ela limita onde o codigo começa e termina, exemplo:

html codigos /html

tag head: utilizada para definir começo e o fim do cabeçalho de um documento html, exemplo:

html head cabeçalho /head /html

tag meta: essa tag traz dados basicos sobre o documento, como autor data de criação, exemplo:

meta Autor, data meta/

tag body: indica para o navegador o corpo do documento HTML, exemplos:

html body parte principal /body /html

tag section: é utilizada para definir seções em um documento separando conteudos, exemplo:

section titulo /section>

tag div: funciona como um contêiner e define um bloco inteiro de conteudo, util para organizar elementos na página web

div Todo, qualquer e quantos elementos do HTML você queira colocar aqui. /div
